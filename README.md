# YAS Advertising – Corporate Website

A modern, high-performance corporate website built for **YAS Advertising (Yas Agency)** using React.  
This website was developed specifically for **YASAgency for advertising and marketing services**.  
The project represents a complete digital presence for a media and advertising agency, focusing on scalability, clean architecture, performance optimization, and premium UI implementation.

---

## 🌐 Live Demo

🔗 **Website:** https://yasagency.info/  
🌍 Hosted on **Hostinger** with custom domain configuration and optimized production deployment.

---

## 📌 Project Overview

YAS Advertising required a professional corporate website that reflects a strong brand identity and premium positioning in the media industry.

The goal was to build a fast, responsive, scalable website with a clean UI system and modern frontend architecture that can be easily extended in the future.

This project was fully designed and developed from scratch using a component-based architecture and a mobile-first approach.
